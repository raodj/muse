This wizard guides you through the process of adding a project to the workspace. A project is a collection of:
  • The set of header and source files associated with a given MUSE simulation on a given server.

  • The make file to be used for compiling the source files to generate a simulation executable.

  • The simulation executable present on a server to be run each time a simulation-job is submitted.

  • The directory to be used for storing job files and outputs from each simulation-run.
